# AI Agents Operating Guide

## Purpose
This repository contains AI agents designed to support energy deal sourcing, capital matching, and project evaluation.

## Agent Roles
- Capital Matcher → connects projects with funding sources
- Deal Screener → evaluates project viability
- Risk Analyzer → identifies financial & operational risks

## Workflow
1. Submit project details
2. Score & classify opportunity
3. Match with capital providers
4. Generate outreach brief

## Standards
- Maintain accuracy and credibility
- Flag unrealistic projections
- Protect partner relationships
